Keamanan & Verifikasi

1. Limit Harian — Monitor 24 Jam Grup Anda

Fungsi utama: Mendeteksi dan membatasi aktivitas pengiriman pesan di grup — sangat cocok untuk memantau userbot / akun promosi yang terus menerus mengirim pesan dalam jumlah besar.

🎯 Mode Deteksi (2 Pilihan)
  • Detect Akun → Hitung jumlah pesan per user ID.
  • Detect Pesan → Identifikasi pesan YANG SAMA dan kelompokkan per pengirim.
⚙️ Batas Maksimal & Reset

Atur limit berapa kali per hari (contoh: 200x). Reset otomatis setiap pukul 00:00 WIB (Asia/Jakarta) tanpa perlu intervensi.

Jenis HukumanKeteranganDurasi?
🔇 MuteMembisukan user di grupPermanen / durasi (detik, menit, jam, hari)
👢 KickMengeluarkan user dari grupLangsung (tanpa durasi)
🚫 BanMelarang user join kembaliPermanen / durasi (auto unban)
🗑️ DeleteMenghapus pesan userDelete Lasted (pesan terakhir) / Delete All (semua pesan hari ini)
📊 Statistik Real-time

Menampilkan daftar user yang melanggar limit, jumlah pelanggaran per hari, dan status hukuman. Data tetap tersimpan hingga reset harian.

📝 Custom Text Notifikasi

Buat pesan kustom dengan template: {MENTION}, {FULLNAME}, {USERNAME}, {GROUPNAME} dan lainnya.

2. Force Delay — Jeda Waktu Wajib Antar Pesan

Cara kerja: Admin menentukan jeda minimal (contoh: 3 menit). Bot mencatat waktu pesan pertama. Jika user mengirim pesan kedua sebelum jeda habis → hukuman diberikan. Hukuman dilepas otomatis setelah waktu jeda terlewati.

⏰ Format Input Jeda
  • 10 detik / 10s
  • 5 menit / 5m
  • 2 jam / 2h
  • 1 hari / 1d
⚖️ Jenis Hukuman
  • 🔇 Mute (durasi/permanen)
  • 🚫 Ban (durasi/permanen + auto unban)
  • 👢 Kick
  • 🗑️ Delete (Lasted / All — hapus 1 atau 2 pesan terakhir)
📝 2 Jenis Pesan Kustom:
  • Pesan Hukum → Dikirim saat bot menghukum user (mendukung template {RELEASE_TIME}).
  • Pesan Bebas → Dikirim saat hukuman dilepas (template {MENTION}, {FULLNAME} dll).

Setiap pesan bisa diaktifkan/nonaktifkan secara terpisah dengan tombol toggle. Jika nonaktif, bot menggunakan pesan default.

3. Verif Absen — Wajib Absen Setiap Hari

Konsep: Semua anggota grup WAJIB melakukan absen ke bot pada jam yang ditentukan. Tanpa absen → tidak bisa mengirim pesan (kena hukuman).

⏰ Pengaturan Jam Absen

Admin input jam HH:MM (contoh: 07:00 atau 00:30). Jam absen yang diubah akan berlaku mulai besuk (tidak mengganggu absen hari ini).

🔗 Link Unik Absen

Format: https://t.me/namabot?start=absen_KODE_CHATID. Link ditampilkan langsung di menu pengaturan.

📢 Notifikasi & Pinned Message

Pada jam yang ditentukan, bot akan mengirim pesan + memin-nya di grup. Klik link → user langsung tercatat absen dan hukuman dilepas.

Hukuman (bagi yang belum absen)

Mute / Ban / Kick / Delete. Sama seperti fitur lainnya, support durasi & permanen.

📝 Custom Text (2 Jenis)
  • Text Info → Dikirim saat notifikasi absen (mendukung {ABSEN_LINK}, {GROUPNAME}).
  • Text Hukum → Dikirim saat user dihukum karena belum absen (template {MENTION}, {ABSEN_LINK}).

Statistik absen menampilkan siapa saja yang sudah absen hari ini lengkap dengan waktu.
