Kebijakan Privasi

1. Informasi yang Kami Kumpulkan

Kami mengumpulkan informasi yang Anda berikan saat mendaftar, termasuk username, password terenkripsi, dan data Telegram ID Anda.

2. Penggunaan Data

Data Anda digunakan untuk mengelola akun bot clone, memproses transaksi, dan memberikan layanan support.

3. Keamanan Data

Password disimpan dalam bentuk hash menggunakan SHA-256. Data transaksi dienkripsi dan disimpan dengan aman.

4. Berbagi Data

Kami tidak menjual atau membagikan data pribadi Anda kepada pihak ketiga tanpa izin Anda.

5. Cookie

Website ini menggunakan cookie untuk menyimpan sesi login dan preferensi Anda.

6. Perubahan Kebijakan

Kebijakan privasi dapat berubah sewaktu-waktu. Perubahan akan diinformasikan melalui website.

7. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi, hubungi kami di support@fragmentbot.com
